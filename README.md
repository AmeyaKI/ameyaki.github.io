# ameyaki.github.io

Personal site — [ameyaki.github.io](https://ameyaki.github.io). Vite + React + TypeScript.

```bash
nvm use && npm install && npm run dev   # http://localhost:5173
```

Edit content in `src/data/` (`profile.ts`, `activity.ts`, `skills.ts`).

Pushes to `main` deploy via GitHub Actions → GitHub Pages.
